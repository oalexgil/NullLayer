# AI Privacy Policy Generator

Generate GDPR and LGPD compliant privacy policies using AI workflows.

## Features

- GDPR support
- LGPD support
- Markdown output
- PDF-ready formatting
- AI workflow ready

- Prompt system

- Usage examples

- Startups
- AI products

## Requirements

- Claude or OpenAI
- n8n optional

---

## Usage

1. Copy PROMPT_SYSTEM.md into Claude.
2. Send company variables.
3. Generate policy.
4. Export as markdown or PDF.
